Model Evaluation 1
• R-squared or coefficient of determination
• SST – Sum of Squares Total (measures variation from the mean)
• SSE – Sum of squares Error (measures variation of data from
model’s predicted values)
• R-squared = (SST-SSE)/SST; used to explain the degree to which input (or
predictor) variables explain the variation of the output (predicted or target)
variables
• R-squared value: 0-1 (equivalent to 0% to 100%)- how close data is to a
fitted line; R-squared value = 1.0 or 100% is perfect fit. If R-squared is 0.9
then it indicates that 90% of the output variables is explained by the input
variables
• R-squared value – higher indicates a better goodness of fit for the model
• Increased improvement in regression model -> increases in R-squared


--- PAGE 13 ---
• Critique of R-squared
• How to improve R-squared value?
• Add more predictors to the regression model
• The increase is artificial because the predictors are not actually improving
the regression model. This is known as overfitting the model.
• If number of predictors increases -> degree of freedom increases
• Adjusted R-squared
• Adjusted R-squared - R-squared value which indicates how well the
predictors fit a model, and adjusted for the number predictors in the model
• It adjusts the R-squared for predictors that are not significant in a regression
model
• Suppose we add additional input variables to a model and the resultant
adjusted R-squared:
• decreases indicates that the additional input variables are not adding value to the model
• Increases indicates that the additional input variables are adding value to the model
• Adjusted R-squared is always equal or less than R-squared
• https://corporatefinanceinstitute.com/resources/knowledge/other/adjusted-
r-squared/


--- PAGE 14 ---
• F-Statistics
• F-test
• To answer the question “Do the independent variables reliably predict the
dependent variable?”. The p-value is compared to your alpha level (typically
0.05) and, if smaller, you reject the null hypothesis (or accept the alternative
hypothesis); you can conclude “Yes, the independent variables reliably predict
the dependent variable”.
• Compares a model with no predictor (intercept-only model – mean of Y when
X=0) and specified model;
• Hypotheses test
• Null hypothesis – fit of the intercept-only model and your model are equal
• Alternative hypothesis – fit of your model is better than the intercept-only model
• F-statistic for the overall model: example, F value = 387.18, p<0.0001 (reject
null hypothesis, highly significantly different)

Logarithm Regression Model
• Logarithm Regression Model
• Equation is Y = a + b*ln(X)
• X must be greater than 0
• a is the intercept
• b is coefficient
• When b >0, model is increasing
• When b < 0, model is decreasing
• ln – natural log
• used to model situations where growth or decay accelerates rapidly at first and
then slows over time.
• https://courses.lumenlearning.com/ivytech-collegealgebra/chapter/build-a-lo

Exponential Regression Model
• Exponential Regression Model
• Equation is Y = ab X
• a is not equal to 0
• Exponential regression model is used to model situations in which:
• growth begins slowly and then accelerates rapidly without bound (to infinity),
• or where decay begins rapidly and then slows down to get closer and closer to
zero.

Polynomial Regression
• Polynomial regression is a special case of linear regression where we fit a
polynomial equation on the data with a curvilinear relationship between the
target variable and the independent variables (look at the scatterplot).
• Univariate Polynomial Regression - Look at Example 5.1 in lab exercises
• Multivariate Polynomial Regression – Look at Example 5.2 in lab exercises
• Univariate - Degree 2 : Y = intercept +aX1 + bX1 2 + residual (or bias)
• a, b are weights
• X1 is a predictor
• 2 - degree
• Bivariate – Degree 2 : Y = intercept + aX1 + bX2 + cX1 2 + dX2 2 + (eX1X2 – if
there is interaction) + residual (or bias)
• a, b, c, and d are weights
• X1 and X2 are predictors
• 2 - degree
